# wa_direct_chat

Lightweight Flutter app to open WhatsApp chats directly by phone number.

Badges
------

- **License:** MIT
- **Status:** Early development

Overview
--------

`wa_direct_chat` provides a minimal UI to start a WhatsApp conversation with a phone number (international format). It's intended as a small utility or example for integrating `url_launcher` and quick chat flows.

Key features
------------

- Open WhatsApp conversation by entering a phone number
- Supports copy/paste and quick dialing formats
- Small, dependency-light Flutter app

Quickstart
----------

Prerequisites:

- Flutter SDK (stable)
- A device or emulator with WhatsApp installed (for full end-to-end testing)

Clone and run:

```bash
git clone https://github.com/agabaeldon/wa_direct_chat.git
cd wa_direct_chat
flutter pub get
flutter run
```

If you intend to publish or link this repository on GitHub, replace `agabaeldon` with your GitHub username.

Development

- Format: `flutter format .`
- Analyze: `flutter analyze`
- Tests: `flutter test`
